# Presentación del Proyecto: ATEL SISTEMS (Web Corporativa)

## Diapositiva 1: Introducción y Objetivo
* **Proyecto**: Diseño, desarrollo e implantación de la nueva plataforma web corporativa para ATEL SISTEMS.
* **Objetivo**: Digitalizar el catálogo de servicios de ingeniería y telecomunicaciones, priorizando el rendimiento extremo y el diseño responsivo.

## Diapositiva 2: Tecnología Frontend
* **Filosofía**: Vanilla Web (Sin frameworks pesados) para carga instantánea.
* **Stack**: HTML5 Semántico, CSS3 Arquitectónico (Grid, Flexbox), Vanilla JavaScript (ES6+).

## Diapositiva 3: Arquitectura e Interactividad
* **Animaciones**: Integración de librería AOS (Animate On Scroll) mitigando parpadeos.
* **Multi-idioma (i18n)**: Motor propio de traducción instantánea en cliente por JavaScript.
* **Diseño Hero Mosaico**: Composición visual responsiva con Grid de 5 zonas que evita perder proporciones usando object-fit.

## Diapositiva 4: Rendimiento y SEO
* **WPO**: Formato WebP universal (hasta un 80% menos de peso), lazy loading nativo.
* **Meta SEO Local**: Implementación nativa de JSON-LD y Schema.org para negocios locales (Google Maps, Horarios, Teléfonos).

## Diapositiva 5: Servidor e Implantación
* **Hosting**: Plataformas de tipología Static Site Generator (SSG) o JAMStack (Netlify, Vercel, GitHub Pages) o Servidor web tradicional (Nginx/Apache).
* **Entregables**: Despliegue estático 100% frontend. No requiere backend (Node.js, Python) ni bases de datos. Los correos se gestionan mediante servicios serverless como Formspree.
